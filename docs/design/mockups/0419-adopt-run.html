Adopting a single run while parked

You are parked on a snapshot, comparing. cool is the snapshot's wording (dotted), warm is the scene now (solid). You want one region's snapshot wording back without restoring the whole scene. Three gestures, live below — hover the tinted words.

The three facts that decide the shape

There is only one operation

While parked, the live scene already is the warm side — so “keep now” is a no-op. The only act that changes anything is take the snapshot's version here. One verb, not a symmetric pair.

One verb, three shapes

Revert covers all of them: a modification swaps the words, a deletion restores them, an insertion's snapshot version is nothing — so revert drops it. That last case is the (x).

§J is honoured, not amended

The chosen gesture (C) adds no glyph at all — the tinted run's own colour becomes clickable. So §J stands as written; Amendment 4 only names the §G / §I boundary (a click that adopts a region is not typing).

Decision: C — click the version you want

The deciding read is the writer's mental model. Parked and comparing, they aren't thinking “revert” — they're thinking “which of these two is better?” Clicking answers that question directly, on either side, and each click settles the region and quiets it. The fact that keeping the warm side doesn't rewrite the scene is invisible to that model, and doesn't need to surface.

It's also the option that leaves the design language untouched: no glyph, so §J holds exactly as written, and the compare view stays visually a set of coloured runs — now merely clickable. Amendment 4 narrows to the §G / §I carve-out.

The one thing C owes is discoverability — nothing announces the runs are clickable. Proposed default: a one-line first-park hint (“Click a passage to keep that version”) that retires after first use, plus cursor: pointer and the hover ring you can feel above. Easily changed.

Frontend-only: the runs already carry both sides, so adopting a region is a re-projection, not a splice — no backend change. None of this is committed; it's a scratch mockup.
